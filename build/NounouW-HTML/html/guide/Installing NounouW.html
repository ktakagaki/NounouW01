Installing NounouW

Obtain a git client if you don't have one already (for simple use, something like GitHub for Windows or GitHub for Mac may be the best choice). Using git will allow you to use the up-to-date version of NounouW at all times. It will also allow you to rewind to a previous version, in case you want to reproduce a previous figure or analysis exactly.

If you do serious programming, you likely have a git client already, and you likely use the git commands on the command prompt. I personally use TortoiseGit for day-to-day git work on Windows.

The NounouW github repository is at the following address: https://github.com/ktakagaki/nounouw. This repository can be downloaded (i.e. "cloned") to your local harddisk using the git client you set up in the previous step. The repository includes all package code, including a runnable .jar file backend. The repository address (https://github.com/ktakagaki/nounouw) is all you need to set to do a Git "pull."

For those planning to contribute to NounouW, please also download the development repository ( https://github.com/ktakagaki/nounouw_dev ). This development repository is structured as a Wolfram Workbench project, and includes some non-essential support code to facilitate development. The devleopment repository is periodically deployed to the main NounouW repository, and you can think of this as a "release."

When you create a local repository, the easiest way is to put the local repository on the Mathematica Path, so that it can be loaded as an application.
